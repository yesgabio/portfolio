Web site construído em HTML, CSS e JavaScript para o trabalho final da disciplina de web no curso técnico em informática. Primeiro projeto Full Stack, com conexão ao banco de dados relacional, o web site possibilita ao usuário realizar CRUD (Created, Read, Update, Delete) dos produtos.
